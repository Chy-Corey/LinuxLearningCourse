## Vim 编辑器

Vim（Vi IMproved）是 Linux/Unix 系统中最强大、最高效的**键盘驱动型文本编辑器**。它基于“模式”操作，一旦掌握，可极大提升代码编写、配置修改等文本处理效率。本文将带你系统掌握 Vim 的核心用法。

### 一、Vim 是什么？

- **起源**：Vim 是经典编辑器 `vi` 的增强版（Vi Improved），完全兼容 `vi` 命令。
- **特点**：
  - 无需鼠标，全键盘操作
  - 模式化设计（不同模式下按键含义不同）
  - 轻量、快速、资源占用低
  - 支持语法高亮、插件扩展、多窗口等高级功能
- **适用场景**：
  - 编写/修改代码
  - 编辑系统配置文件（如 `/etc/nginx.conf`）
  - 远程服务器上的文本处理（无图形界面时必备）

> 💡 大多数 Linux 发行版已预装 `vim` 或 `vi`。若未安装，可运行：
> ```bash
> # Ubuntu/Debian
> sudo apt install vim
> # CentOS/RHEL
> sudo yum install vim-enhanced
> ```

### 二、Vim 的核心：四种模式

Vim 的一切操作都围绕**模式切换**展开。理解模式是使用 Vim 的关键！

| 模式                         | 别名     | 功能                                     | 如何进入                                                     | 如何退出                                             |
| ---------------------------- | -------- | ---------------------------------------- | ------------------------------------------------------------ | ---------------------------------------------------- |
| **普通模式**(Normal)         | 命令模式 | 执行光标移动、删除、复制、粘贴等编辑命令 | 启动即在此模式，按 `Esc` 从其他模式返回普通模式              | 按 `i`, `a`, `o` 等进入插入模式按 `:` 进入命令行模式 |
| **插入模式**(Insert)         | 编辑模式 | 像普通编辑器一样输入文本                 | 在普通模式下按：`i`（当前光标前插入）`a`（光标后插入）`o`（下一行新行插入） | 按 `Esc` 返回普通模式                                |
| **命令行模式**(Command-line) | 底行模式 | 执行保存、退出、查找、替换、设置等命令   | 在普通模式下按 `:`                                           | 按 `Enter` 执行命令或按 `Esc` 取消                   |
| **可视模式**(Visual)         | 选择模式 | 选中文本区域进行批量操作                 | 在普通模式下按：`v`（字符选择）`V`（整行选择）`Ctrl+v`（块选择） | 按 `Esc` 或执行操作后自动退出                        |

> ✅ **口诀**：  
> **“普通模式做操作，插入模式输文字，命令行里下指令，可视模式选区域。”**

### 三、基本操作流程（新手必看）

#### 1. 打开文件
```bash
vim filename.txt    # 打开或创建文件
vim +10 file.log    # 打开并跳转到第 10 行
```

#### 2. 编辑文件（标准流程）
1. 启动后处于 **普通模式**
2. 按 `i` 进入 **插入模式**，开始输入文字
3. 编辑完成后，按 `Esc` 回到 **普通模式**
4. 按 `:` 进入 **命令行模式**，输入保存/退出命令

#### 3. 保存与退出（在命令行模式下）
| 命令          | 作用               |
| ------------- | ------------------ |
| `:w`          | 保存（write）      |
| `:q`          | 退出（quit）       |
| `:wq` 或 `:x` | 保存并退出         |
| `:q!`         | 强制退出（不保存） |
| `:w filename` | 另存为             |

> ⚠️ 如果文件被修改但未保存，直接 `:q` 会报错，必须用 `:q!` 强制退出。

### 四、普通模式常用命令（高效编辑的核心）

#### 🔁 光标移动
| 命令            | 作用                         |
| --------------- | ---------------------------- |
| `h` `j` `k` `l` | 左、下、上、右（替代方向键） |
| `w` / `b`       | 跳到下一个/上一个单词开头    |
| `0`（零）       | 行首                         |
| `$`             | 行尾                         |
| `gg`            | 文件开头                     |
| `G`             | 文件末尾                     |
| `5G`            | 跳到第 5 行                  |

#### ✂️ 编辑操作
| 命令      | 作用               |
| --------- | ------------------ |
| `x`       | 删除当前字符       |
| `dd`      | 删除整行           |
| `5dd`     | 删除 5 行          |
| `yy`      | 复制（yank）当前行 |
| `5yy`     | 复制 5 行          |
| `p`       | 粘贴到光标后       |
| `P`       | 粘贴到光标前       |
| `u`       | 撤销（undo）       |
| `Ctrl+r`  | 重做（redo）       |
| `.`（点） | 重复上一次编辑操作 |

> 💡 示例：  
> `d$` → 删除从光标到行尾  
> `dw` → 删除一个单词  
> `ciw` → **改写**当前单词（Change Inner Word）

### 五、命令行模式常用命令

| 命令               | 作用                                        |
| ------------------ | ------------------------------------------- |
| `:set nu`          | 显示行号                                    |
| `:set nonu`        | 取消行号                                    |
| `:syntax on`       | 开启语法高亮                                |
| `/keyword`         | 向下搜索关键词（按 `n` 下一个，`N` 上一个） |
| `?keyword`         | 向上搜索                                    |
| `:%s/old/new/g`    | 全文替换所有 `old` 为 `new`                 |
| `:5,10s/foo/bar/g` | 替换第 5~10 行中的 `foo` 为 `bar`           |
| `:e otherfile.txt` | 在当前窗口打开另一个文件                    |

### 六、实用技巧与组合

#### 1. 快速注释/取消注释（结合可视模式）
```vim
Vjjj      " 选中 4 行（V + j×3）
:s/^/# / " 在每行开头加 #
```

#### 2. 多文件编辑
```vim
:e file2.txt    " 切换到另一个文件
:bnext / :bprev " 切换缓冲区
:ls             " 列出所有打开的文件
```

#### 3. 分屏操作
| 命令             | 作用             |
| ---------------- | ---------------- |
| `:sp filename`   | 水平分屏         |
| `:vsp filename`  | 垂直分屏         |
| `Ctrl+w h/j/k/l` | 在分屏间移动光标 |

### 七、个性化配置（`.vimrc`）

Vim 的行为可通过 `~/.vimrc` 文件自定义。推荐基础配置：

```vim
" 显示行号
set number
" 语法高亮
syntax on
" 高亮搜索结果
set hlsearch
" 智能缩进
set autoindent
set smartindent
" 使用 4 个空格代替 tab
set tabstop=4
set shiftwidth=4
set expandtab
```

> 创建配置文件：
> ```bash
> touch ~/.vimrc
> vim ~/.vimrc   # 粘贴上述内容
> ```

### 八、学习建议

1. **不要怕按错**：Vim 有完善的撤销机制（`u`）
2. **强制自己不用鼠标**：初期慢，后期飞快
3. **每天练习 10 分钟**：肌肉记忆是关键
4. **使用内置教程**：终端运行 `vimtutor`（官方交互式教学，强烈推荐！）

### 九、总结

> **Vim = 模式切换 + 组合命令 + 键盘流**

- **新手阶段**：记住 `i` → 编辑，`Esc` → 退出编辑，`:wq` → 保存退出
- **进阶阶段**：熟练使用 `dd`/`yy`/`p`/`u`/`/` 等命令
- **高手阶段**：利用宏、正则、插件实现自动化编辑

> 💡 **终极目标**：让手指记住命令，眼睛专注内容，大脑思考逻辑——这才是 Vim 的真正魅力。
